# LMS App (Workspace)

This folder contains the full source code for the LMS CodingGita Clone project.

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```

In another terminal:

```bash
npm run server
```

Frontend: `http://localhost:5173`  
Backend health: `http://localhost:5000/api/health`
